# Insight - AI 智能学情洞察系统项目总结

## 1. 项目愿景
Insight 旨在从传统的“被动查题工具”进化为“主动学习与教研工作台”。通过 DeepSeek AI 引擎、全方位 CRM 学生管理及权威级中考英语知识考点库，为教师和学生提供具备高阶逻辑推理能力的智能化教学闭环。

## 2. 核心技术栈
- **UI 框架**: Jetpack Compose (1.5+) - 采用物理弹簧动效 (`Spring`) 与响应式流式布局。
- **AI 引擎**: DeepSeek V3 API (支持 SSE 流式协议与打字机动效)。
- **图像处理**: CameraX + ML Kit (实现智能题目锁定与像素级裁剪)。
- **数据层**: Room (复杂多表关系映射) + Preferences DataStore (配置持久化)。
- **办公解析**: Apache POI (支持 Excel、Word 及压缩包的结构化数据提取)。
- **渲染层**: Markwon (高性能 Markdown 原生渲染，支持表格与 HTML 标签)。

## 3. 核心功能模块 (Milestones)

### 3.1 权威级中考英语考点库与星系知识图谱 (Zhongkao Hub & Galaxy Graph)
- **结构化知识图谱 3.0 (实时联动版)**: 按照“词法、时态、句法”三大底层板块构建了 24+ 核心考点字典，并以**星系拓扑结构**呈现，支持**渐进式认知展开**。
- **物理仿真交互**: 图谱具备实时的**Spring 弹簧动力学**，支持手势缩放、平移及 AI 的**焦点高亮模式**。
- **实时掌握度探测**: 图谱节点颜色与发光强度直接与数据库 `StudentMastery` 挂钩。通过 `InsightViewModel` 实现 UI 状态切换与图谱视觉效果的秒级同步。
- **十年真题精解**: 动态索引 2016-2024 全国中考原题，提供包含标准解析、易错点预警及全句翻译的三维诊断模型。

### 3.2 智能教研大脑 (The AI Brain)
- **流式实时解析**: 基于 SSE 技术，让 AI 的逻辑推理过程以“可见即所得”的打字机方式呈现。
- **多维学情诊断**: 能够基于 Room 沉淀的细颗粒度数据，自动生成包含雷达图指标及复习建议的报告。
- **智能备课助手**: 支持自然语言生成 Markdown 教案初稿，并能自动关联班级高频错题。

### 3.3 学生管理枢纽 (CRM System)
- **多源批量导入**: 兼容教师日常办公习惯，支持 Excel 表格、Word 表格及包含多文档的 ZIP 压缩包一键灌入。
- **全生命周期档案**: 建立了“学生-题目-报告”的深度数据关联，支持 360° 查看学生能力演变趋势。

### 3.4 工业级拍摄裁剪系统
- **全边框自由选区**: 拍照后支持四角及四边的任意拉伸调节，实现像素级精准裁剪。
- **沉浸式视觉转场**: 采用“圆形扩散”视觉特效，优化硬件初始化时的视觉连续性。

- **沉浸式开屏叙事 (The Fluid Calligraphy Splash) 3.0**: 深度融合 Android 12+ Splash API 与 Compose 动效引擎，实现了比肩 Apple 级的动态物理仿真动画。抛物线下坠的小球在触地瞬间产生挤压与拉伸 (Squash & Stretch)，随后化作笔尖，通过实时生成的 SVG 贝塞尔曲线行云流水地写出 "nsight"。最后，第二颗小球自由落体砸中首字母 `ı`，触发底层物理引擎（Spring Physics）计算的弹性谐振与相位错位弹跳 (Overlapping Action)，呈现出极具生命力的视觉张力。
- **考点掌握度全生命周期管理**: 为每个知识点引入了“待复习、练习中、已掌握”三态管理模型。支持 `DataStore` 持久化记录，并与详情页的 Apple 级状态切换开关及 Haptic 触感反馈深度整合，构建了完整的学习进度反馈闭环。

## 4. 界面设计语言 (Design Language)
- **视觉风格**: 现代简约 + 玻璃拟态。采用精致投影 (`elevation`)、高对比度配色方案（SageGreen & InkBlue）。
- **排版与渲染引擎**: 全面集成并深度调优 `Markwon`，实现教研级 Markdown 解析，支持复杂表格强行去缩进与多行代码块容错。定制化背景高亮 (`TallHighlightSpan`) 实现精细到词级别的重点标注。
- **布局体系**: 首页模块化程度进一步提升。顶部问候语标准化，实现了“用户中心”式的精准触达。
- **交互动效**: 全量应用 `PathMeasure` 与物理仿真动画。
    - **知识点图标指纹化**: 为 24+ 核心考点设计了唯一的视觉图标，增强了认知的直觉关联。
    - **智能文本提取**: 首页预览自动提取核心介绍，过滤 Markdown 噪音。
    - **分级弹性容器**: 全面应用 `AnimatedVisibility` 与 `Spring` 动画参数，首页分级列表展开柔和自然。
- **导航语义**: 标准 Android 路由导航，支持全屏转场缩放动效。

## 5. 未来展望
- **课堂互动插件**: 增加随机点名、限时阅读计时器等实时授课工具。
- **自适应学习路径**: 根据学生在“考点通”中的浏览与真题练习数据，AI 自动调节知识星图的节点权重。
