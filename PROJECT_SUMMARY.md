# Insight - AI 智能学情洞察系统项目总结

## 1. 项目愿景
Insight 旨在从传统的“被动查题工具”进化为“主动学习系统”。通过 AI 诊断引擎、沉浸式知识图谱和多维数据反馈，为学生提供温情且专业的专属辅导体验。

## 2. 核心技术栈
- **UI 框架**: Jetpack Compose (1.5+)
- **架构模式**: MVVM + Hilt 依赖注入
- **图像处理**: CameraX + Google ML Kit (OCR)
- **底层绘制**: Compose Canvas (用于无限星空画布、贝塞尔连线、雷达图)
- **状态管理**: Kotlin Coroutines + Flow + SharedViewModel (Activity 作用域)
- **持久化**: SharedPreferences (用户信息与偏好设置)

## 3. 已实现里程碑
### 3.1 导航与交互系统
- **动态 Dock 栏**: 具有物理弹性的 Gooey 菜单，支持 Z 轴缩放转场。
- **全局状态共享**: 统一 ViewModel 作用域，确保设置、主页、诊断报告信息 100% 同步。
- **状态持久化**: 实现用户名（默认“用户”）、班级（默认“一年级一班”）及角色（默认“老师”）的本地存储。

### 3.2 沉浸式知识图谱 (The Starfield)
- **无限画布引擎**: 基于 Canvas 的底层绘制，支持双指缩放、平移拖拽。
- **动态星系渲染**: 
    - 节点根据掌握度（Mastery Level）自动呈现绿色（已掌握）、黄色（学习中）或琥珀色（薄弱点）。
    - 连线采用贝塞尔曲线，并根据关联关系自动高亮。
- **下钻交互**: 点击节点平滑升起 BottomSheet 深度解析考点。

### 3.3 全局视觉优化
- **完美暗色模式**: 彻底清理硬编码颜色，全量适配 `MaterialTheme.colorScheme`，消除暗色模式下的“黑色字体”不可见问题。
- **字体系统**: 规范化 `Type.kt`，所有样式动态继承主题色。

## 4. 正在进行的突破：AI 导师诊断报告
- **打字机动态寄语**: 模拟 AI 实时思考的反馈。
- **生长式雷达图**: 具备路径追踪动画的五维能力评估。
- **错因热力图**: 借鉴 GitHub 风格的连续学习强度可视化。
- **AI 处方卡片**: 基于 RAG 架构预留的靶向突破入口。
