# Insight - AI 智能学情洞察系统项目总结

## 1. 项目愿景
Insight 旨在从传统的“被动查题工具”进化为“主动学习系统”。通过 AI 诊断引擎、沉浸式知识图谱和多维数据反馈，为用户提供温情且专业的专属辅导体验。

## 2. 核心技术栈
- **UI 框架**: Jetpack Compose (1.5+)
- **图像处理**: CameraX + ML Kit (实现智能题目锁定与像素级裁剪)
- **底层绘制**: Compose Canvas (无限星空画布、动态选区遮罩、雷达图)
- **持久化层**: Room 数据库 + Preferences DataStore
- **渲染引擎**: WebView + HTML5 (用于专业 A4 PDF 报告导出)

## 3. 已实现里程碑
### 3.1 工业级拍摄裁剪系统
- **智能选区交互**: 实现了拍照后“全边框可拖拽”的编辑模式，支持四角缩放、四边拉伸及整体平移。
- **视觉引导**: 初始采用 85% 宽度的“大视野”取景框，配合 ML Kit 实时分析实现题目自动锁定建议。
- **无缝转场**: 恢复了相机按钮的圆形扩散动画，并优化了导航时序，彻底消除了硬件加载时的界面闪烁。

### 3.2 沉浸式知识图谱 (The Starfield)
- **漫游引擎**: 基于 Canvas 的无限画布，支持手势缩放与平移。
- **数据驱动**: 节点颜色根据 Room 数据库中的掌握度实时变化。

### 3.3 AI 导师与 PDF 导出
- **诊断报告**: 具备打字机动效、生长式雷达图及错因热力图。
- **WYSIWYG 导出**: 所见即所得的 PDF 预览与定制化导出流程。
