# Insight - AI 智能学情洞察系统项目总结

## 1. 项目愿景
Insight 旨在从传统的“被动查题工具”进化为“主动学习与教研工作台”。通过 DeepSeek AI 引擎、全方位 CRM 学生管理及模块化教案系统，为教师和学生提供具备高阶逻辑推理能力的智能化教学闭环。

## 2. 核心技术栈
- **UI 框架**: Jetpack Compose (1.5+) - 采用响应式流式布局与高级动画。
- **AI 引擎**: DeepSeek V3 API (支持 SSE 流式协议与打字机动效)。
- **图像处理**: CameraX + ML Kit (实现智能题目锁定与像素级裁剪)。
- **数据层**: Room (复杂关系映射) + Preferences DataStore (配置持久化)。
- **办公解析**: Apache POI (支持 Excel、Word 及压缩包的结构化数据提取)。
- **渲染层**: Markwon (高性能 Markdown 原生渲染)。

## 3. 核心功能模块 (Milestones)

### 3.1 智能教研大脑 (The AI Brain)
- **流式实时解析**: 基于 SSE 技术，让 AI 的逻辑推理过程以“可见即所得”的打字机方式呈现。
- **多维学情诊断**: 能够基于 Room 沉淀的细颗粒度数据，自动生成包含雷达图指标、错因归因及复习建议的专业报告。
- **智能备课助手**: 支持自然语言生成教案初稿，并能自动关联班级高频错题库。

### 3.2 学生管理枢纽 (CRM System)
- **多源批量导入**: 兼容教师日常办公习惯，支持 Excel 表格、Word 表格及包含多文档的 ZIP 压缩包一键灌入。
- **全生命周期档案**: 建立了“学生-题目-报告”的深度数据关联，支持 360° 查看学生学习轨迹与能力演变趋势。

### 3.3 工业级拍摄裁剪系统
- **全边框自由选区**: 拍照后支持四角及四边的任意拉伸调节，实现像素级精准裁剪。
- **无缝转场动画**: 采用“圆形扩散”视觉特效，彻底修复硬件初始化时的黑屏或闪烁问题。

### 3.4 沉浸式知识图谱
- **数据驱动画布**: 基于 Compose Canvas 打造的无限星空，节点亮度与颜色随数据库中的掌握度动态变化。

## 4. 界面设计语言 (Design Language)
- **视觉风格**: 现代简约 + 玻璃拟态。采用精致投影 (`elevation`) 与大圆角 (`24.dp`) 营造模块化呼吸感。
- **交互逻辑**: 标准 Android 导航语义，返回键固定于 TopAppBar 左上角，转场动效采用系统级 Scale + Fade。

## 5. 未来展望
- **课堂互动插件**: 增加随机点名、限时阅读计时器等实时授课工具。
- **知识点自动更新**: AI 授课后自动反馈并调节知识星图中的预期掌握度。
