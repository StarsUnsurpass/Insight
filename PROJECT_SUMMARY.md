# Insight - AI 智能学情洞察系统项目总结

## 1. 项目愿景
Insight 旨在从传统的“被动查题工具”进化为“主动学习系统”。通过 AI 诊断引擎、沉浸式知识图谱和多维数据反馈，为用户提供温情且专业的专属辅导体验。

## 2. 核心技术栈
- **UI 框架**: Jetpack Compose (1.5+)
- **架构模式**: MVVM + Hilt 依赖注入 + Shared ViewModel (Activity 作用域)
- **底层绘制**: Compose Canvas (用于无限星空图谱、贝塞尔连线、生长式雷达图)
- **持久化层**: 
    - **Jetpack Room**: 存储关系型业务数据（识题记录、知识节点、诊断报告）。
    - **Jetpack DataStore**: 存储用户偏好设置（姓名、班级、角色、主题、震动反馈）。
- **渲染引擎**: WebView + HTML5/CSS3 (用于完美的 A4 PDF 分页导出)
- **图像处理**: CameraX + Google ML Kit (OCR)

## 3. 已实现里程碑
### 3.1 工业级数据架构
- **持久化升级**: 废弃 SharedPreferences，全量迁移至 DataStore，实现异步类型安全存储。
- **响应式数据库**: 建立 Room 数据库，支持 Flow 实时流转，确保“数据库一变，UI 自动刷新”的单点真相架构。
- **状态无缝同步**: 统一全局 ViewModel，确保设置项在首页、图谱和报告页瞬间同步，且应用重启后不重置。

### 3.2 沉浸式知识图谱 (The Starfield)
- **交互漫游引擎**: 支持双指缩放、单指平移及上帝视角回正。
- **动态渲染**: 节点颜色根据掌握度（Mastery Level）实时反馈（绿色掌握/琥珀色薄弱）。

### 3.3 AI 导师诊断报告
- **多维可视化**: 打字机特效寄语、生长式五维雷达图、学习强度热力图。
- **错因深度剖析**: 提供基于大模型归因的占比分析与教研建议。

### 3.4 专业级 PDF 导出系统 (WYSIWYG)
- **所见即所得预览**: A4 比例实时预览，支持标题、主题色和内容维度的即时配置。
- **混合渲染技术**: 利用系统 Print API 与 HTML 模板，确保 PDF 排版精美、分页自然且文本可复制。
