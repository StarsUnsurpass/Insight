# Insight - AI 智能学情洞察系统项目总结

## 1. 项目愿景
Insight 旨在从传统的“被动查题工具”进化为“主动学习系统”。通过 AI 诊断引擎、沉浸式知识图谱和多维数据反馈，为用户提供温情且专业的专属辅导体验。

## 2. 核心技术栈
- **UI 框架**: Jetpack Compose (1.5+)
- **架构模式**: MVVM + Hilt 依赖注入 + Shared ViewModel
- **底层绘制**: Compose Canvas (无限星空画布、贝塞尔连线、生长式雷达图)
- **渲染引擎**: WebView + HTML/CSS (用于完美的 A4 PDF 分页导出)
- **状态管理**: Kotlin Coroutines + Flow
- **持久化**: SharedPreferences (用户信息与偏好设置本地化)

## 3. 已实现里程碑
### 3.1 导航与交互系统
- **动态 Dock 栏**: 具有物理弹性的 Gooey 菜单，支持 Z 轴缩放转场。
- **全局状态共享**: 统一 ViewModel 作用域，确保设置、主页、诊断报告信息 100% 同步。
- **状态持久化**: 实现用户名（默认“用户”）、班级（默认“一年级一班”）及角色（默认“老师”）的本地持久化。

### 3.2 沉浸式知识图谱 (The Starfield)
- **交互漫游**: 基于 Canvas 的底层绘制，支持双指缩放、平移拖拽及上帝视角回正。
- **下钻交互**: 点击节点平滑升起 BottomSheet，展示“易错陷阱分析”与“靶向特训”。

### 3.3 AI 导师诊断报告
- **多维可视化**: 包含打字机特效的 AI 寄语、生长式雷达图及 GitHub 风格的学习强度热力图。
- **错误归因**: 深度剖析错误背后的逻辑原因（概念盲区、审题偏差等）。

### 3.4 专业级 PDF 导出系统
- **WYSIWYG 预览**: 实时模拟 A4 纸张比例的报告预览，支持标题、主题色、内容维度的即时配置。
- **混合渲染技术**: 采用 Compose 驱动配置，后台通过隐藏 WebView 渲染 HTML 模板，利用系统 Print API 生成完美分页、文本可复制的 PDF 文档。
- **安全分享**: 集成 FileProvider 实现 PDF 报告的一键微信/钉钉分享。
