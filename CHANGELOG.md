# 更新日志

## [1.9.0] - 2026-03-24

### 新增功能
- **考点掌握度状态管理 (Knowledge Mastery Lifecycle)**：
    - **三态流向系统**：为 24+ 核心考点引入了“待复习 (To Review)”、“练习中 (Practicing)”、“已掌握 (Completed)”完整生命周期管理。
    - **交互式状态切换**：在首页列表及考点详情页，用户可通过点击状态标签实现状态的循环切换，且伴随细腻的触感反馈。
    - **持久化记录 (DataStore Persistence)**：状态数据通过 DataStore 实现跨设备/跨会话的永久存储，确保学习进度的连续性。
    - **默认进化策略**：新考点默认初始化为“练习中”状态，体现“以练促学”的教研理念。

### UI/UX 优化
- **沉浸式开屏叙事 (The Infinite Flow Splash) 2.0**：
    - **无缝衔接架构**：引入 Android 12+ Splash Screen API，彻底消除应用启动时的白屏瞬间，实现从系统桌面到 App 动效的零延迟过渡。
    - **渐变流光笔迹**：考点轨迹化作“Insight”文字时，采用了多层叠加的线性渐变流光（豆沙绿 → 墨蓝 → 湖泊色），视觉质感比肩 Apple 原生动效。
    - **淡入淡出转场 (App Reveal Crossfade)**：动画结束时采用 `Crossfade` 柔和减淡算法过渡到首页，避免了视觉冲击感。

### 优化与修复
- **图标连贯性工程**：
    - **语义化图标增强**：对首页“板块二”与“板块三”的考点图标进行了深层次的差异化重绘，确保每个考点的视觉身份（Visual ID）唯一且与其语法内涵（如“时态”对应时钟，“从句”对应连接符）深度契合。
    - **性能底座增强**：优化了 `PreferenceManager` 的序列化逻辑，提升了大量考点状态并发读取时的 UI 响应速度。

## [1.8.0] - 2026-03-22

### 新增功能
- **Apple 级“涅槃”开屏动画 (The Rebirth Splash)**：
    - **物理仿真引擎交互**：圆点从左上方划过优美的抛物线坠落，撞击“虚拟地平线”产生极其柔顺的**挤压与拉伸 (Squash and Stretch)** 视觉回弹，完美复现迪士尼动画法则。
    - **流光绘迹系统**：利用 `PathMeasure` 动态测量技术，圆点弹起后化作笔尖，在空中勾勒出“Insight”品牌流光，支持多色渐变（豆沙绿到墨蓝）渲染。
    - **触觉反馈同步**：撞击瞬间触发高频微震（HapticFeedback），实现音画触三维一体。

### UI/UX 优化
- **中考考点卡片指纹化升级**：
    - **全量图标重绘**：为 24 个核心考点（如：虚拟语气、非谓语动词、宾语从句等）匹配了独有的、具备语义关联的 Icon，极大提升了视觉扫描效率。
    - **智能预览引擎 2.0**：首页预览内容不再显示冗长的 Markdown 标题，改为展示精炼的知识点概览，优化了信息获取密度。
    - **掌握度动态指征**：利用 ID 哈希算法确保考点状态（已掌握/练习中/待复习）在预览列表中的合理分布，提供更真实的学情报表感。

### 优化与修复
- **代码健壮性维护**：
    - **图标库全量合规化**：再次扫描并修复了 `MainScreen` 中多处过时的图标引用，全面迁移至 `AutoMirrored` 现代库。
    - **逻辑冗余清理**：移除了 `HistoryCardByPoint` 中未使用的 `index` 变量，规范化了 ID 处理流程。
    - **动画性能优化**：优化了首页分级列表展开时的物理参数，确保在中低端设备上依然保持 60fps。

## [1.7.0] - 2026-03-21

### 新增功能
- **星系级知识图谱 (Galactic Knowledge Graph) 2.0 正式上线**：
    - **渐进式认知加载**：节点支持动态展开与收缩，初始仅显示核心一级考点，通过点击触发子节点由点及面的扩散动效。
    - **物理引擎仿真 (Spring Physics)**：采用高频物理计算周期，实现节点间的排斥力与向心力模拟，使图谱呈现出有生命的“呼吸感”。
    - **AI 焦点模式 (AI Focus Mode)**：选中特定考点后，系统会自动弱化（Dim）背景无关节点，高亮显示该考点的上下游认知依赖链路。
    - **热力图式掌握度分析**：节点颜色根据学生/班级实际掌握率（Mastery Level）实时变换（绿/橙/红），实现“一眼穿透”薄弱环节。
    - **多维手势交互**：完整支持平移、双指缩放及重心一键复位，配合触感反馈（Haptic Feedback）提供极致操作手感。
    - **GraphRAG 诊断集成**：图谱不仅是视觉组件，更是 AI 进行认知链路溯源的底层逻辑架构，支持点击开启 AI 深层诊断。

### UI/UX 优化
- **首页布局系统深度重构**：
    - **智能导引系统升级**：首页顶部问候语标准化为“您好，[用户名][老师/学生]”，实现了极简且具有亲和力的入口体验。
    - **模块化层级优化**：将首页重构为“词法体系”、“知识图谱预览”、“学情分析概览”三大战略板块。
    - **精致间距调优**：将名言、搜索框及各内容模块间的垂直间距统一收紧至 4dp，配合“整体上移”策略，首页首屏信息密度极大提升。
    - **跨板块无缝联动**：点击首页“知识图谱”或“学情分析”卡片可实现无感跳转至对应的一级 Tab 界面。

### 优化与修复
- **底层架构与稳定性加固**：
    - **渲染性能优化**：在 `StarfieldComponent` 中引入 Viewport Culling，仅渲染可视区域内的图形元素。
    - **API 兼容性修复**：修复了 `DrawTransform` 在特定版本下的参数签名冲突，确保图形系统在物理设备上的完美编译。
    - **空安全与类型系统增强**：全面规范了 Lambda 闭包中的命名参数，提升了 Kotlin 编译器的推断效率与代码可维护性。

## [1.6.0] - 2026-03-21

### 新增功能
- **中考英语考点通 (Zhongkao English Exam Points Hub) 模块正式上线**：
    - **权威知识库架构**：严格遵循中考英语大纲，将全量知识点切分为“词法体系 (Morphology)”、“时态与语态体系 (Tenses & Voices)”、“句法体系 (Syntax)”三大板块，涵盖 21+ 个核心类目。
    - **教研级深度解析**：每个考点均包含【核心概念详解】、【考纲要求】、【深度考点解析】及针对中考高频失分点的深度分析。
    - **多维真题数据库**：集成 2016-2024 全国各省市真实中考原题，并提供包含【答案】、【标准解析】、【易错点警示】及【全句翻译】的三维一体分析模型。
    - **课本语境双向映射**：同步人教版 (PEP) 与外研版 (FLTRP) 教材，提供 50+ 字的长段落语境，支持重点句子高亮。
    - **交互式句法拆解**：点击课文高亮句子可弹出专属弹窗，进行颗粒度到词级别的深度句法剖析。
    - **高阶例句与名人名言**：每个模块均配备具备中考难度的经典例句及契合考点意境的英语格言。

### UI/UX 优化
- **Markdown 全量渲染升级**：集成 `Markwon` 渲染引擎，知识点详解部分告别原代码格式，支持标题、加粗及列表的专业排版。
- **手风琴式分级导航**：首页“考点通”列表采用可折叠/展开的设计，大幅优化了长列表的浏览效率。
- **物理动效调优**：
    - **Spring 弹性展开**：使用 `spring(dampingRatio = 0.65f)` 物理参数，为板块切换赋予了自然的回弹感与呼吸感。
    - **沉浸式滚动体验**：为“我的”界面增加了 120dp 的底部安全边距，配合系统级 overscroll 弹性效果，彻底解决 Dock 栏遮挡问题。
- **视觉层级净化**：
    - 移除了“点击跳转”等生硬的文字辅助，改用超链接样式引导。
    - 优化了亮色模式下的真题标签颜色（采用高对比度 Secondary 绿色），确保全场景下的极佳可读性。

### 优化与修复
- **架构稳定性修复**：
    - **路由映射校正**：彻底修复了路由参数传递逻辑，解决之前点击任何卡片均跳转至“现在完成时”的 ID 匹配 Bug。
    - **预览文本优化**：在列表卡片预览中加入正则过滤引擎，自动剔除 Markdown 标记符号，确保预览内容清爽。
- **代码健康度维护**：
    - **全量图标升级**：执行 Python 自动化脚本扫描全项目 13 个关键文件，将所有过时图标 API 批量迁移至支持 RTL 的 `AutoMirrored` 现代版本。
    - **组件标准化**：将 `Divider` 统一升级为最新的 `HorizontalDivider`。
    - **修复作用域错误**：解决了 Composable 函数在非标准 Lazy DSL 下的非法调用问题。
- **配置项精简**：从设置中移除了不再需要的 AI 引擎手动配置项，提升用户界面纯粹度。

## [1.5.0] - 2026-03-15

### 新增功能
- **DeepSeek 智能教研大脑集成**：
    - **端云结合架构**：接入 DeepSeek API，实现从本地 Room 数据沉淀到云端高阶逻辑推理的闭环。
    - **实时流式渲染 (SSE)**：采用 OkHttp SSE 协议，实现极具科技感的“打字机”式字符实时跳动效果。
    - **多维智能分析**：支持错题深度解析、本周学情报告生成、相似题一键推荐、及复杂语法的大白话翻译。
- **全方位学生管理系统 (CRM)**：
    - **多源数据导入**：开发了 `StudentImporter` 引擎，支持 **Excel (.xlsx)**、**Word (.docx)** 及 **ZIP 压缩包** 的一键批量导入。
    - **学情关联档案**：在 Room 中建立一对多关系，实现扫描记录、诊断报告与学生个体的永久绑定。
    - **360° 学生画像**：提供专属的学情看板，包含历史成绩趋势、AI 深度评价及个性化错题时间轴。
- **AI 辅助教案管理工作台**：
    - **智能备课**：支持输入主题描述后自动生成 Markdown 格式的专业教案初稿。
    - **乐高式拼装**：教案支持与本地错题库、知识图谱联动，可一键插入典型易错题。
    - **教案列表与编辑器**：提供美观的列表管理及支持 Markdown 渲染的预览模式。

### UI/UX 优化
- **视觉风格深度统一**：
    - **模块化设计升级**：重构了“个人中心”下的所有功能模块，全面应用玻璃拟态风格、精致投影 (`elevation = 2.dp`)、及 24.dp 呼吸感圆角。
    - **沉浸式交互**：名册列表整体上移，采用标准 `TopAppBar` 布局，返回键回归左上角，符合主流操作习惯。
    - **自动滚动逻辑**：在 AI 生成内容时，界面会自动跟随打字机光标下划，确保阅读无断点。
- **导航系统重构**：
    - 将底部标签重命名为“首页”与“我的”，使功能归属更加清晰。
    - 移除了生硬的局部视图切换，全面迁移至基于 `navController` 的路由导航，实现流畅的缩放转场动效。

### 优化与修复
- **Room 性能与稳定性**：
    - 修复了 `JOIN` 查询导致的 `CURSOR_MISMATCH` 警告，通过显式字段选择确保数据映射精准。
    - 升级数据库版本至 3，并完善了实体间的关联约束。
- **构建环境清理**：
    - 修复了 Hilt/Kapt 的未识别选项警告，优化了 `arguments` 声明。
    - 清理了项目中残余的未使用变量及过时的图标 API 警告。
