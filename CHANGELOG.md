# 更新日志

## [1.5.0] - 2026-03-15

### 新增功能
- **DeepSeek 智能教研大脑集成**：
    - **端云结合架构**：接入 DeepSeek API，实现从本地 Room 数据沉淀到云端高阶逻辑推理的闭环。
    - **实时流式渲染 (SSE)**：采用 OkHttp SSE 协议，实现极具科技感的“打字机”式字符实时跳动效果。
    - **多维智能分析**：支持错题深度解析、本周学情报告生成、相似题一键推荐、及复杂语法的大白话翻译。
- **全方位学生管理系统 (CRM)**：
    - **多源数据导入**：开发了 `StudentImporter` 引擎，支持 **Excel (.xlsx)**、**Word (.docx)** 及 **ZIP 压缩包** 的一键批量导入。
    - **学情关联档案**：在 Room 中建立一对多关系，实现扫描记录、诊断报告与学生个体的永久绑定。
    - **360° 学生画像**：提供专属的学情看板，包含历史成绩趋势、AI 深度评价及个性化错题时间轴。
- **AI 辅助教案管理工作台**：
    - **智能备课**：支持输入主题描述后自动生成 Markdown 格式的专业教案初稿。
    - **乐高式拼装**：教案支持与本地错题库、知识图谱联动，可一键插入典型易错题。
    - **教案列表与编辑器**：提供美观的列表管理及支持 Markdown 渲染的预览模式。

### UI/UX 优化
- **视觉风格深度统一**：
    - **模块化设计升级**：重构了“个人中心”下的所有功能模块，全面应用玻璃拟态风格、精致投影 (`elevation = 2.dp`)、及 24.dp 呼吸感圆角。
    - **沉浸式交互**：名册列表整体上移，采用标准 `TopAppBar` 布局，返回键回归左上角，符合主流操作习惯。
    - **自动滚动逻辑**：在 AI 生成内容时，界面会自动跟随打字机光标下划，确保阅读无断点。
- **导航系统重构**：
    - 将底部标签重命名为“首页”与“我的”，使功能归属更加清晰。
    - 移除了生硬的局部视图切换，全面迁移至基于 `navController` 的路由导航，实现流畅的缩放转场动效。

### 优化与修复
- **Room 性能与稳定性**：
    - 修复了 `JOIN` 查询导致的 `CURSOR_MISMATCH` 警告，通过显式字段选择确保数据映射精准。
    - 升级数据库版本至 3，并完善了实体间的关联约束。
- **构建环境清理**：
    - 修复了 Hilt/Kapt 的未识别选项警告，优化了 `arguments` 声明。
    - 清理了项目中残余的未使用变量及过时的图标 API 警告。

## [1.4.0] - 2026-03-14

### 新增功能
- **智能拍摄选区系统升级**：
    - **全边框自由交互**：拍摄后选区支持四角缩放及四边独立拖拽调节。
    - **大视野取景引导**：采用 85% 宽度的大框设计，大幅提升题目对焦体验。
- **无缝视觉转场**：
    - 恢复并优化了相机按钮的“圆形扩散”过渡动画。
    - 修复了跳转拍摄界面时的界面闪烁问题。
