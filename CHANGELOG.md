# 更新日志

## [1.3.0] - 2026-03-14

### 新增功能
- **全量升级工业级存储架构**：
    - **Jetpack Room**：建立关系型数据库，完整支持识题历史、知识图谱及学情报告的本地化。
    - **Jetpack DataStore**：废弃 SharedPreferences，采用异步、类型安全的 DataStore 存储用户偏好。
    - **单点真相 (SSOT)**：重构 Repository 层，通过 Flow 实现数据库变动到 UI 渲染的响应式闭环。
- **高级 PDF 导出系统 (WYSIWYG)**：
    - **实时导出预览**：新增 A4 比例预览页面，支持即时调整报告标题、主题色及模块显示。
    - **系统级打印集成**：接入 Android Print API，支持通过系统对话框“保存为 PDF”或直接打印。
- **设置系统增强**：
    - 增加默认值配置（姓名：用户，班级：一年级一班，角色：老师）。
    - 优化角色选择逻辑，支持“老师/同学”身份的实时切换与 UI 深度适配。

### 优化与修复
- **状态同步优化**：统一 Activity 级别的 ViewModel 作用域，彻底解决跨页面信息不同步问题。
- **视觉体验升级**：
    - 修复暗色模式下“黑色字体”不可见及背景异常变白的问题。
    - 完善首页搜索交互，支持一键清除与动态状态切换。
- **极致清理编译警告**：
    - 彻底修复了 unused variables, unresolved references 等 10 余项编译器警告。
    - 配置 kapt 选项成功压制 Hilt 生成代码的日志干扰。
