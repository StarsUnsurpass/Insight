# Insight - Android 智能分析与“温柔的效率”

Insight 是一款基于 Android 16 (SDK 36) 开发的智能学习辅助应用，专注于通过视觉识别技术帮助用户高效批改、解析题目，并构建个性化的知识星图。

## 核心设计理念：温柔的效率 (Gentle Efficiency)

*   **极简视觉**：采用莫兰迪色系（米白、燕麦、黛蓝），降低视觉疲劳，让使用过程如翻阅手账般从容。
*   **沉浸式解析**：解析内容采用分步卡片流展示，配合交错动效，让复杂的知识点层层递进。
*   **智能手感**：
    *   **可拖动选题框**：支持四角、四边及全局拖动，像修图一样精准圈选题目。
    *   **玻璃拟态交互**：采用毛玻璃特效与微动效，打造高级感与流畅度并存的原生体验。

## 关键技术特性

*   **Jetpack Compose 全量构建**：利用原生声明式 UI 实现流畅的转场与交互动画。
*   **CameraX 深度集成**：支持扫码线呼吸特效、闪光反馈以及高精度图像捕获。
*   **动态知识星图**：使用自定义 Canvas 绘制的可交互知识图谱，通过浮动动效展示知识点间的逻辑连接。
*   **现代 Android 架构**：基于 Hilt 的依赖注入与 Clean Architecture。

## 如何运行

1. 使用 Android Studio Ladybug 或更高版本（推荐 Preview 版本以支持 SDK 36）打开项目。
2. 确保已通过 SDK Manager 安装 Android 16 (SDK 36) 环境。
3. 在 `gradle.properties` 中已配置 `android.suppressUnsupportedCompileSdk=36` 以抑制 SDK 版本警告。
4. 运行应用，体验“温柔的效率”带来的流畅扫描与解析过程。
